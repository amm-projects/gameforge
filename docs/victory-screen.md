# Pantalla de Victoria

## Añadido: v0.21.0 (2026-06-08)

## ¿Cómo funciona?

Cuando el jugador llega a la meta (`goal`) aparece una pantalla de victoria con fondo oscuro y dos botones:

- **Retry** — reinicia el nivel desde el principio.
- **Stop** — cierra el runtime y vuelve al editor.

El juego se detiene por completo (enemigos, gravedad y colisiones se pausan) mientras la pantalla está visible.
